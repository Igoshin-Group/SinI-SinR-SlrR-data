This folder contains the fluorescence images used in this work and scripts processing these images. 

/images     -- microscopic images used in this work
    /gfp    -- fluorescence images of GFP intensity
    /rfp    -- fluorescence images of RFP intensity
    /open   -- open phase microscopic images used for segmentation
    /signal -- processed data for each cell in each image
    /tapA-production-estimate -- data used to estimate tapA production rate
        /gfp    -- fluorescence images of GFP intensity
        /open   -- open phase microscopic images used for segmentation
